# Voice Emotion Chatbot Backend

This is the backend for the Voice Emotion Chatbot. It accepts audio files, detects the speaker's emotion, optionally transcribes the audio to text, and generates a response based on the emotion.

---

## Features

- Detects emotions: happy, sad, neutral
- Optional speech-to-text transcription
- Handles both WAV and MP3 audio files
- Robust error handling
- JSON responses ready for frontend integration
